1.1 冒泡排序

代码思想：

  • 每一轮排序，通过移动确定一个最大的数字在最左边或者最右边，n个数字经历n-1轮就可以排序成功
  • 嵌套循环：外层循环控制循环的次数，内层循环进行遍历
  • 外层循环每循环一次，循环次数增加一，内层循环需要比较的元素数量就会减少一
  • 讲解思想：小鲤鱼吐泡泡，大的泡泡先浮上去
  • 语法：python交换两个数字 a,b = b,a

1.2 选择排序

代码思想：

  • 每一轮循环确定一个最小的数字或者最大的数字在最左边
  • 通过遍历将本轮第一个数字（基准值）和剩下来的所有数字进行比较，如果发现有数字比本轮第一个数字小。交换该数字和基准数字，继续遍历，直到本轮结束
  • 外循环控制本轮的基准索引值，内循环用来遍历剩下来下来的所有数字

1.4 快速排序

代码思想：

  • 找出基准元素，取列表中间值为基准元素，如果中间由两个数字，取右边数字为基准元素
  • 创建两个小列表，从做往右遍历除了基准元素的所有元素，小于基准元素的数字放在左边小列表里，大于或者等于基准元素的数据放在右边列表里面
  • 使用递归的递把所有列表进行一直拆分，使用递归的归把处理后的列表进行合并
  • 思想：分治思想

1.5 归并排序

代码思想：

  • 从哪里开始分割？ 从中间开始分割，如果有奇数个元素，左边列表为元素数量少的列表
  • 分到什么情况下不用再分？ 分到列表中元素的数量只剩下一个的时候不需要进行分割
  • 合并的规则是什么？ 创建一个新列表，两个新指针，两个指针指向左右两个小列表，开始比较所指元素的大小，小的元素插入到新列表中，然后指针右移继续和另外一个指针所指的元素比较，直到某个小列表遍历结束，把另外一个列表剩下的所有元素插入到新列表中
  • 整个过程中更体现了什么思想？ 分治思想

图解函数：

  • gray # 灰度值是0-255之间的一个数字

  • unit = 256/length # 把灰度值按照列表的长度分成length份，每一份都对应了一个灰度区间，比如length = 10，unit = 25.6 ,每一个字符可以表示26个灰度值，索引值为0的字符灰度区间为0-25，索引值为1的字符区间为26-51...............…….

  • int(gary/unit) #求一下当前颜色的灰度值在第几个字符的灰度区间

  • ascii_char[int(gray/unit)] #根据求出来的第几个字符，从列表拿出该字符
